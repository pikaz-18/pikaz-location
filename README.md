## Introduction

供中国地区使用的定位插件

## Features

* 支持h5定位、ip定位、经纬度查询地址、省市区三级联动列表搜索

ps：由于浏览器限制，http域名的网页使用h5定位可能会出现问题，如定位不准、禁止定位等，如果想要使用精准定位的话，最好使用https域名
